# 🇷🇺 Aprende Ruso — A1 a C2

App de aprendizaje de ruso con niveles A1, A2, B1, B2, C1 y C2.  
Hecho por **María de los Milagros Esther Calderón**

## Características
- 6 niveles MCER (A1 → C2) con sistema de desbloqueo
- Lecciones de idioma, gramática y cultura rusa
- Exámenes de 15 preguntas con temporizador
- Gráficas de progreso (radar + barras)
- Frases de inspiración en ruso
- Curiosidades culturales

## Cómo correr localmente

```bash
npm install
npm run dev
```

## Cómo subir a Vercel

### Opción A — CLI (más rápido)
```bash
npm install -g vercel
npm run build
vercel
```

### Opción B — GitHub + Vercel web
1. Subir esta carpeta a un repositorio de GitHub
2. Ir a [vercel.com](https://vercel.com) → New Project
3. Importar el repositorio
4. Vercel detecta Vite automáticamente → Deploy ✓

## Tecnologías
- React 18
- Vite 5
- Recharts
